# 网推律所全链路一体化SaaS产品需求文档V1\.0

## 一、文档概述

### 1\.1 文档目的

本文档明确网推律所全链路一体化SaaS产品的完整功能需求、业务规则与链路逻辑，作为产品研发、测试、客户交付的核心依据，产品融合蘑菇投前端获客销转能力、金助理后端案管财务能力、Alpha法律专业工具能力，补全全竞品缺失的网推专属合规风控体系，实现「获客\-销转\-办案\-合规\-财务\-服务」全闭环。

### 1\.2 适用范围

本需求覆盖产品全部核心业务模块与支撑底座，适配广告投放驱动、实行「邀谈分离」流水线模式的网推律所，兼容中小团队与规模化连锁律所。

### 1\.3 目标读者

产品研发团队、测试团队、律所运营负责人、销售团队、实施交付团队。

### 1\.4 产品核心定位

面向网推律所的全链路一体化合规SaaS平台，以营销获客为入口、销转提效为核心、办案标准化为支撑、合规风控为底线、财务核算为闭环，打造法律服务行业唯一覆盖公域到交付全流程的数字化系统。

### 1\.5 目标用户与角色定义

|角色名称|核心职责|
|---|---|
|投放岗|广告账户管理、素材投放、投放效果优化|
|邀约岗|线索触达、客户邀约到所、邀约数据统计|
|谈案岗|客户接待、谈单签约、商机跟进|
|办案律师/助理|案件办理、任务执行、文书撰写、客户沟通|
|行政/财务岗|审批流程、收费开票、分润核算、台账管理|
|律所管理者|全链路数据查看、经营决策、合规管控|
|C端客户|查看案件进度、提交材料、在线咨询、评价服务|

### 1\.6 核心业务全流程总链路

公域广告投放 → 线索自动留资入库 → 加微引流私域 → 线索分配邀约岗 → 电话邀约到所 → 谈案岗接待签约 → 立案进入办案系统 → 标准化办案流程 → 结案归档 → 财务分润核算 → 客户服务与口碑沉淀 → 全链路数据回流优化投放

## 二、核心功能需求（按模块逐个功能明确）

### 模块1：全域投放与获客营销系统

#### 1\.1 广告账户统一管理

- **功能描述**：支持主流广告平台账户统一接入与管控，无需切换多平台后台

- **业务规则**：

    1. 原生对接巨量引擎、百度营销、腾讯广告、快手广告四大平台，支持单平台多账户绑定

    2. 支持账户分组、权限分配，仅投放岗/管理员可操作账户，其他角色仅可查看数据

    3. 账户余额、消耗数据实时同步，余额低于阈值自动触发预警通知

- **前置条件**：完成广告平台API授权

- **输出**：统一账户列表、实时消耗数据

#### 1\.2 投放计划批量管控

- **功能描述**：跨平台批量管理广告计划、单元、创意

- **业务规则**：

    1. 支持计划批量启停、预算调整、出价修改、复制迁移

    2. 支持按平台、账户、案由、素材类型筛选计划

    3. 计划操作日志全留存，可追溯操作人与操作时间

- **前置条件**：广告账户已授权绑定

- **输出**：计划操作结果、批量操作日志

#### 1\.3 全链路转化归因

- **功能描述**：打通广告投放与后端转化数据，实现全链路ROI核算

- **业务规则**：

    1. 支持线索、加微、邀约到所、签约回款四级转化事件回传广告平台

    2. 可按渠道、账户、计划、创意、关键词维度，统计消耗、线索量、线索成本、加微率、签约率、ROI

    3. 转化数据T\+1自动更新，支持实时数据查询

- **前置条件**：广告平台授权、后端CRM转化数据同步

- **输出**：全链路转化报表、分维度ROI数据

#### 1\.4 投放素材效能管理

- **功能描述**：统一管理全平台投放素材，统计素材转化效果

- **业务规则**：

    1. 支持图文、短视频、直播脚本等素材分类存储、标签管理

    2. 自动关联素材对应投放计划，统计素材的曝光、点击、转化、ROI数据

    3. 支持素材效果排行，自动标记高转化素材与低效素材

- **前置条件**：广告账户授权、素材上传入库

- **输出**：素材效果排行榜、素材转化明细

#### 1\.5 AI营销内容生成

- **功能描述**：AI批量生成营销类内容，支撑投放与私域运营

- **业务规则**：

    1. 支持输入案由、核心卖点，自动生成短视频脚本、朋友圈文案、直播话术、科普图文

    2. 内置网推律所高频案由（婚姻、劳动、债务、交通事故）内容模板

    3. 生成内容自动进入素材库，可直接用于投放或私域发布

- **前置条件**：法律AI中台能力可用

- **输出**：AI生成的营销内容文稿

#### 1\.6 营销内容合规预审

- **功能描述**：所有对外发布的营销内容前置合规审核，拦截违规内容

- **业务规则**：

    1. 自动识别内容中的夸大宣传、包胜诉承诺、绝对化用语、违规收费承诺等违规点

    2. 审核结果分为「通过、需修改、禁止发布」三类，标注违规位置与修改建议

    3. 审核不通过的素材无法绑定投放计划，无法发布至私域

- **前置条件**：内容上传至系统、合规风控模块可用

- **输出**：合规审核结果、违规标注与修改建议

#### 1\.7 公域账号矩阵管理

- **功能描述**：统一管理律所全平台公域账号，调度内容发布

- **业务规则**：

    1. 支持抖音、快手、视频号、公众号账号绑定与分组管理

    2. 支持内容排期、定时发布，发布记录全留存

    3. 账号粉丝、点赞、咨询数据统一统计

- **前置条件**：公域账号授权绑定

- **输出**：账号矩阵数据报表、内容发布记录

### 模块2：公私域连接器与SCRM私域运营系统

#### 2\.1 多场景活码管理

- **功能描述**：生成多场景引流活码，实现公域线索高效沉淀企微/个微

- **业务规则**：

    1. 支持企微活码、个微活码、群活码三类，可自定义分流规则

    2. 支持按轮询、负载、地域、案由分配对应销售/邀约账号

    3. 每个活码绑定唯一渠道标识，可追溯来源

- **前置条件**：企业微信/个人微信接入系统

- **输出**：活码图片、活码渠道配置

#### 2\.2 引流渠道全链路追踪

- **功能描述**：统计每个引流渠道的全链路转化效果

- **业务规则**：

    1. 每个活码/推广链接对应独立渠道，统计渠道的扫码量、加微量、加微率、开口率、邀约率、签约率

    2. 支持渠道分组、数据对比，自动标记高转化渠道

    3. 渠道数据与投放系统打通，自动关联对应广告计划

- **前置条件**：活码/推广链接已创建、线索数据同步

- **输出**：渠道转化效果报表

#### 2\.3 客户标签体系管理

- **功能描述**：多维度客户标签管理，支撑分层运营与精准触达

- **业务规则**：

    1. 标签分为自动标签与手动标签两类

    2. 自动标签：来源渠道、广告素材、咨询案由、意向等级、浏览行为、跟进阶段，系统自动打标

    3. 手动标签：客户特征、需求细节、风险等级，由运营/销售手动添加

    4. 标签全系统同步，CRM、SCRM、办案系统共享标签数据

- **前置条件**：客户信息入库

- **输出**：客户标签画像、标签分组列表

#### 2\.4 企微侧边运营助手

- **功能描述**：聊天侧边栏展示客户信息与工具，提升沟通效率

- **业务规则**：

    1. 侧边栏展示客户全景档案、历史跟进记录、标签信息、来源渠道

    2. 内置标准化合规话术库、素材库，支持一键发送

    3. 聊天过程中可直接创建跟进任务、预约提醒，同步至CRM

- **前置条件**：企业微信接入、客户信息已同步

- **输出**：侧边栏客户信息、话术素材调用结果

#### 2\.5 私域触达工具集

- **功能描述**：批量私域触达工具，支撑客户培育与唤醒

- **业务规则**：

    1. 支持1V1定时群发、朋友圈统一发布、社群SOP运营

    2. 群发可按标签、客户分组筛选，发送记录全留存

    3. 朋友圈支持统一排期、多账号同步发布，数据统计

- **前置条件**：企微/个微接入、客户标签体系可用

- **输出**：触达任务执行结果、发送数据统计

#### 2\.6 聊天全量存档管理

- **功能描述**：企微/个微聊天记录全程留存，支撑合规质检与客户交接

- **业务规则**：

    1. 文字、图片、语音、视频、文件全类型聊天记录永久存档

    2. 支持按客户、员工、时间、关键词检索聊天记录

    3. 聊天记录自动同步至合规风控模块，触发智能质检

    4. 员工离职无法删除聊天记录，管理员可完整导出

- **前置条件**：企微/个微接入系统

- **输出**：聊天记录存档、检索结果

### 模块3：线索中台与谈案转化CRM

#### 3\.1 全渠道线索归集

- **功能描述**：全渠道线索统一入库，形成唯一线索池

- **业务规则**：

    1. 自动归集：广告平台表单、私信、企微加微、直播间留资线索自动同步入库

    2. 手动归集：支持手动录入、Excel批量导入转介绍/线下线索

    3. 自动去重：按手机号、微信号去重，重复线索自动合并，保留全部来源信息

    4. 线索入库自动生成唯一客户ID，全系统通用

- **前置条件**：广告平台对接、私域渠道对接

- **输出**：统一线索池、客户唯一档案

#### 3\.2 线索前置利益冲突初查

- **功能描述**：线索入库自动触发利益冲突检索，规避接案合规风险

- **业务规则**：

    1. 自动比对客户姓名、企业名称与系统内历史客户、对方当事人库

    2. 检索结果分为「无冲突、疑似冲突、明确冲突」三类

    3. 明确冲突的线索无法分配，仅管理员可查看处理

- **前置条件**：系统内已有客户/案件数据库

- **输出**：利冲检索结果、冲突标记

#### 3\.3 智能线索分配引擎

- **功能描述**：自动分配线索至对应岗位，适配邀谈分离模式

- **业务规则**：

    1. 支持按地域、案由、岗位负载、人员专业标签自定义分配规则

    2. 原生适配邀谈分离：线索先分配邀约岗，邀约成功到所后自动流转至谈案岗

    3. 分配结果实时通知对应员工，支持手动改派，改派日志全留存

- **前置条件**：员工账号已配置、分配规则已设置、线索通过利冲初查

- **输出**：线索分配结果、分配通知

#### 3\.4 线索公海池管理

- **功能描述**：未跟进/无效线索统一回收，二次分配利用

- **业务规则**：

    1. 线索分配后，超过预设时长（默认24小时，支持自定义）未跟进，自动回收至公海池

    2. 支持员工主动退回线索至公海池，退回需填写原因

    3. 公海池线索可重新分配或由员工主动领取，领取次数可限制

- **前置条件**：线索已分配、未跟进超时/主动退回

- **输出**：公海池线索列表、回收/领取记录

#### 3\.5 客户全景档案管理

- **功能描述**：整合客户全维度信息，形成唯一全景档案

- **业务规则**：

    1. 档案包含：基础信息、来源渠道、广告素材、全部通话录音、聊天记录、跟进记录、历史案件、标签信息

    2. 全系统数据自动同步，办案、财务、合规模块的客户数据实时更新至档案

    3. 支持权限管控，仅负责员工与管理员可查看完整档案

- **前置条件**：客户ID已生成、各模块数据同步

- **输出**：客户全景档案页

#### 3\.6 邀约岗专属管理

- **功能描述**：邀约岗专属工作台，支撑批量邀约作业

- **业务规则**：

    1. 工作台展示待跟进线索、今日邀约任务、已邀约列表

    2. 支持一键外呼，通话自动录音、自动存档至客户档案

    3. 可创建到所预约，填写预约时间、案由、客户需求，预约信息自动同步至对应谈案岗

    4. 统计邀约量、接通率、邀约率、到所率核心指标

- **前置条件**：线索已分配至邀约岗

- **输出**：邀约任务列表、通话录音、预约记录

#### 3\.7 谈案岗专属管理

- **功能描述**：谈案岗专属工作台，支撑谈单与商机跟进

- **业务规则**：

    1. 工作台展示今日到所客户、待跟进商机、签约列表

    2. 自动同步邀约岗填写的客户信息、需求、预约记录，无需重复录入

    3. 支持录入谈案记录、商机阶段更新、报价方案管理

    4. 签约后可一键发起立案申请，客户与商机信息自动同步至办案系统

- **前置条件**：客户已邀约到所、分配至谈案岗

- **输出**：谈案记录、商机阶段、立案申请

#### 3\.8 谈案标准化SOP

- **功能描述**：标准化谈案流程，规范谈案动作，降低合规风险

- **业务规则**：

    1. 预置高频案由谈案SOP，包含风险告知、需求确认、方案讲解、报价、签约等强制节点

    2. 未完成强制节点（如诉讼风险告知），无法推进至签约阶段

    3. 支持自定义SOP节点与要求

- **前置条件**：SOP模板已配置

- **输出**：SOP节点完成记录、流程进度

#### 3\.9 谈案AI合规辅助

- **功能描述**：谈案过程实时合规辅助，规避违规表述

- **业务规则**：

    1. 通话/聊天过程中，实时识别违规表述（包胜诉、100%成功、违规承诺），即时弹窗提醒

    2. 自动匹配对应案由的合规话术、风险告知模板，辅助谈案人员应答

    3. 违规记录自动同步至合规风控模块，生成预警

- **前置条件**：通话/聊天接入系统、合规AI能力可用

- **输出**：实时合规提醒、违规预警记录

#### 3\.10 客户资产交接管理

- **功能描述**：员工离职/调岗时，完整交接客户资产

- **业务规则**：

    1. 支持一键移交指定员工的全部客户、线索、商机

    2. 移交内容包含：客户档案、聊天记录、通话录音、跟进记录、待办任务

    3. 移交后原员工失去客户查看权限，操作日志全留存

- **前置条件**：管理员发起交接申请

- **输出**：客户交接结果、交接日志

#### 3\.11 AI营销工作手机适配

- **功能描述**：配套硬件手机与系统原生打通，实现全链路留痕

- **业务规则**：

    1. 手机端微信聊天、通话记录自动回传系统，员工无法删除

    2. 支持敏感操作预警：删除好友、私发联系方式、转账，实时通知管理员

    3. 手机端可查看线索、跟进客户、外呼，数据与PC端实时同步

- **前置条件**：营销手机绑定系统、员工账号绑定

- **输出**：手机端数据回传、操作预警记录

### 模块4：标准化案件办案管理系统

#### 4\.1 案件立案管理

- **功能描述**：签约后案件线上立案，完成案件信息归档

- **业务规则**：

    1. 支持谈案端一键发起立案，自动带入客户信息、案由、签约金额，无需重复录入

    2. 立案需完成正式利益冲突检索、委托合同上传、办案律师指派

    3. 立案审批通过后，生成唯一案件编号，正式进入办案流程

    4. 立案信息自动同步至财务系统、合规系统、C端客户端口

- **前置条件**：客户已签约、立案申请提交、利冲检索通过

- **输出**：案件档案、案件编号、立案通知

#### 4\.2 办案SOP模板管理

- **功能描述**：配置标准化办案流程模板，统一办案标准

- **业务规则**：

## 四、全链路需求衔接性整体校验

### 4\.1 前端获客\-销转链路衔接校验
